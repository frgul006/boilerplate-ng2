The footer stuff

So let's begin by preparing the containers and getting that footer fixed!

html {
height: 100%;
}

body {
min-height: 100%;
}

Percentage height is relative to the parent and if parent doesn't have a defined height, it won't take effect.
By setting it on both html and body, we can make sure they take up (at least) the full height of the browser window.

Next, we tell the body element to use flexbox to position its direct descendants.

body {
display: flex;
flex-direction: column;
min-height: 100%;
}

What we are basically saying with this is: Hey, body! Show your children in a column!
